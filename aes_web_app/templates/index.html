Hướng dẫn sử dụng:

  1. Nhập mã khóa (có thể là bất kỳ chuỗi nào)
  2. Chọn file cần mã hóa hoặc giải mã
  3. Nhấn nút "Mã hóa" hoặc "Giải mã"
  4. File kết quả sẽ được tải xuống tự động
Lưu ý: Để giải mã file, bạn phải sử dụng đúng mã khóa đã dùng để mã hóa.
